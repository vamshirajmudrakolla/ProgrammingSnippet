Variables

  • One of the main concepts in programming is variables. They are your best friends. You will deal with them all the time. You will use them to store information. They will represent your data input.

  • Let’s say you want to have a variable x that is equal to the value of 5 and then ask the computer to tell you the value of that variable. Type x equals 5. Write x and then execute. And here’s the result – 5.
  • An alternative way to execute the instruction that will provide the value we assigned to y would be to use the print command. If we say “print y”, the machine will simply execute this command and provide the value of y as a statement.
